# HomeWork

## Лаба 1
Реализовано простое Flask-приложение для получения ip через два разных апи. Создано 2 функции для запроса и 3 эндпоинта. Один выводит  главную страницу с результатом в виде html разметки, второй в виде json, и третий проверяет здоровье для кубернетиза.

## Лаба 2
Реализован Dockerfile
docker-compose.yml - в нем создано 4 сервиса. Два из них это приложение с разными апи, и прометеус и графана для сборо и визуализации метрик (я выводил количество запросов и проверку работает ли приложение). Так же в код главного приложения добавил сбор метрик через prometheus_flask_exporter.
